# Developer Portfolio

A professional developer portfolio website featuring a clean, minimalist aesthetic inspired by Claude.ai, with integrated "developer touches" like a Terminal UI, Code Snippet sections, and a Dark/Light mode toggle.

## Features
- **Claude.ai Aesthetic:** Warm off-white backgrounds and serif typography for a sophisticated feel.
- **IDE Dark Mode:** Switch to a sleek dark theme for a developer-focused look.
- **Terminal UI:** Animated hero section featuring a bash-like interface.
- **GitHub Integration:** Dynamically fetches project details (stars, description, language) for `symphony` and `IronCopilot`.
- **Code Snippet Styling:** Services and skills presented as syntax-highlighted code blocks.
- **Responsive Design:** Optimized for all screen sizes.

## Local Development
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:5173`.

## Deployment
This project is configured for easy deployment to GitHub Pages:
1. Ensure your code is pushed to GitHub.
2. Run `npm run deploy`.
This will build the project and push the `dist` folder to the `gh-pages` branch.
